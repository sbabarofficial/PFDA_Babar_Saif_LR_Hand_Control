# Maya Importable L/R Hand Control Rig

## Description
A modular, importable left/right hand control rig for Autodesk Maya that includes animator‑friendly attributes for posing and keyframing. This tool streamlines the animation workflow by giving artists a ready‑to‑use hand rig that can be dropped into any character setup.

## Features
- **Modular L/R Hand Control**
  - A clean, self‑contained rig file for each hand that can be imported into any Maya scene without breaking existing rigs.
- **Animator‑Friendly Control Attributes**
  - Custom attributes for curl, spread, twist, and individual finger controls, implemented through driven keys and attribute editor.
- **Zeroed, Clean Controls**
  - Controls are frozen, oriented properly, and ready for keyframing with no cleanup required.
- **Compatibility With Existing Rigs**
  - Designed to be parented or constrained to any character’s wrist joint with minimal setup.

## Challenges
- Researching how to structure rigs for easy import/export without breaking connections.
- Learning how to get the program from python to a simple command that can be run in maya's own python console.
- Ensuring the rig works across different Maya versions and character scales.

## Outcomes

**Ideal Outcome:**  
A fully modular L/R hand rig with polished controls, pose presets, clean naming conventions, and documentation. Animators can import it into any character and immediately begin keyframing expressive hand poses.

**Minimal Viable Outcome:**  
A single hand rig with basic curl/spread controls that imports cleanly and allows animators to keyframe essential hand movements.
